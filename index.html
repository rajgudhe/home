Projects

Multi-level dilated residual network for biomedical image segmentation

A novel multi-level dilated residual neural network was proposed as an extension of the classical U-Net architecture for biomedical image segmentation. The new model replaces convolutional blocks with multi-level dilated residual blocks and incorporates non-linear multi-level residual blocks into skip connections.

Area-based breast percentage density estimation in mammograms using weight-adaptive multitask learning

A deep learning architecture that estimates breast density in mammograms using a weight-adaptive multitask learning approach was proposed. The model simultaneously segments the breast and dense tissues and estimates breast density. It was evaluated on an independent set of 7500 craniocaudal and mediolateral oblique-view mammograms from Kuopio University Hospital, Finland. The multitask approach outperformed multitask U-net and a fully convolutional neural network in terms of F-score by an average of 2.88% and 9.78%, respectively. Breast density values estimated by the model correlated strongly with radiologists' assessments, with a Pearson's correlation of r=0.90. The approach improved segmentation accuracy, reduced time and effort needed to estimate density values, and decreased inter- and intra-reader variability.
